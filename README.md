# ScoreBoard
This simple web page provides a scoreboard.

### HOW TO INSTALL & RUN

```
$ git clone https://github.com/jungsikchoi/ScoreBoard.git
$ cd ScoreBoard
```

```
$ sudo apt-get install python-virtualenv
$ virtualenv venv
$ . venv/bin/activate
$ pip install Flask
```

```
$ python score_board.py
```
